### 定义字体
```css
font-family: "微软雅黑";  /* 字体名字 */
src: url(#);  /* 这个url里面填写字体地址 */
```

### 字体样式

```css
font-style: italic;  /* 斜体 */
```

### 字体装饰

```css
text-decoration: line-through;  /* 下划线 */
text-decoration: underline;  /* 下划线 */
text-decoration: none;  /* 去掉下划线 */
text-decoration: underline overline;  /* 下划线和上划线 */
text-decoration: underline overline line-through;  /* 下划线、上划线和删除线 */
```

### 字体大小

```css
 font-size: 30px;  /* 字体大小 */
```

### 字体粗细

```css
font-weight: 700;  /* 字体加粗 */
```
