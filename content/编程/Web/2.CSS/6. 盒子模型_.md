## 盒子边距

| 外边距 | marging |
| ------ | ------- |
| 内边距 | padding |

### 使用方法

```css
margin: 0 auto;  		/* 居中 */
margin-top: 10px;  		/* 上边距 */
margin-bottom: 10px;  	/* 下边距 */
margin-left: 10px;  	/* 左边距 */
margin-right: 10px; 	/* 右边距 */
margin: 10px 20px;  	/* 上下边距和左右边距 */
margin:  10px 20px 30px 40px;  /* 上、右、下、左 顺时针*/
```

## 盒子边框

```css
border-radius: 1px solid red;border-radius: 1px solid red; /* 圆角，分别为像素，样式，颜色、支持rgb等 */
```
