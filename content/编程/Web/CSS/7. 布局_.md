## 布局元素

```css
display: block; /* 块级元素，独占一行*/

display: inline; /* 行内元素，和其他元素在一行显示 */

display: inline-block; /* 行内块元素，但是可以设置宽度和高度等样式 */

display: none; /* 隐藏元素 */
```

## 弹性布局

```css
display: flex; /* 弹性布局 */

flex-direction: row; /* 横向排列 */
flex-direction: column; /* 纵向排列 */

```

### 弹性方向

```css
justify-content: flex-start; /* 水平左对齐 */
justify-content: center; /* 水平居中 */
justify-content: flex-end; /* 水平右对齐 */
justify-content: space-between; /* 水平两端对齐 */
justify-content: space-around; /* 水平两端对齐，中间间距相等 */

align-items: flex-start; /* 垂直顶部对齐 */
align-items: center; /* 垂直居中 */
align-content: flex-end;  /* 垂直底部对齐 */
align-content: space-between;  /* 垂直两端对齐 */
align-content: space-around;  /* 垂直两端对齐，中间间距相等 */
```



### 设置占用比例

```css
flex: 1;
```

### 设置间距

```css
gap: 10px; /* 间距 */
```

## 网格布局

```css
display: grid; /* 网格布局 */
grid-template-columns: repeat(3, 1fr);      /* 3列，每列宽度相等 */
grid-template-columns: 100px 200px 300px;   /* 3列，每列宽度相等 */

grid-auto-rows: 100px; /* 自动行高 */
grid-template-rows: repeat(3, 1fr);         /* 3行，每行高度相等 */
grid-template-rows:100px 100px 300px;       /* 3行，每行高度相等 */

grid-gap: 10px; /* 间距 */
```
