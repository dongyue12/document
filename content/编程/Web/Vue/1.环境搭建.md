### 环境搭建

[教程推荐](https://www.bilibili.com/video/BV1Za4y1r7KE?spm_id_from=333.788.player.switch&vd_source=4b2a3aa4c1475ef90a44549a323239ae)

[官方文档](https://cn.vuejs.org/guide/quick-start.html)

```node
# 查看@vue/cli版本，确保@vue/cli版本在4.5.0以上
vue --version

# 安装或者升级你的@vue/cli 
npm install -g @vue/cli

# 执行创建命令，不能大写
npm create vue@latest

cd 到项目目录然后
npm i 
npm run dev

项目运行起来后
输入o可以用默认浏览器打开
输入q退出
输入r重启
```

### 跟换镜像

```
# 查看当前 npm 源
npm config get registry

淘宝镜像源
npm config set registry https://registry.npmmirror.com

阿里云镜像源
npm config set registry https://npm.aliyun.com

腾讯云镜像源
npm config set registry https://mirrors.cloud.tencent.com/npm/

切换回官方源
npm config set registry https://registry.npmjs.org/
```

#### 使用 nrm 快速管理镜像源

#### nrm 是一个方便的 npm 源管理工具，支持快速查看和切换镜像源。

```
安装 nrm
npm install -g nrm
查看可用镜像源

nrm ls
切换镜像源 例如切换到淘宝镜像：

nrm use taobao
测试镜像速度

nrm test

```

#### 注意事项

```
切换后建议运行 npm config get registry 确认是否生效。
淘宝 npm 镜像已于 2021 年正式迁移至 npmmirror.com 域名
```
