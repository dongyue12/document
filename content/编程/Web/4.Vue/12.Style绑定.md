###  Style 绑定

在 Vue 中，通过 `v-bind:style`（简写 `:style`）可以动态地绑定内联 CSS 样式。语法为 **对象形式**，属性名使用 **camelCase**（如 `fontSize`）或 **kebab-case 加引号**（如 `'font-size'`）。

#### 1. 内联绑定（直接写死样式）

直接在模板中传入一个样式对象。

vue

```
<template>
  <h3>1. 内联绑定</h3>
  <div :style="{ color: 'red', fontSize: '15px' }">
    红色文字，15px 字号（直接写死）
  </div>
</template>
```



#### 2. 样式切换（三元表达式）

结合三元表达式，根据数据状态动态切换样式值。

vue

```
<template>
  <div>
    <h3>2. 样式切换</h3>
    <button @click="toggleColor">切换颜色</button>
    <div :style="{ color: isActive ? 'red' : 'blue' }">
      当前颜色：{{ isActive ? '红色' : '蓝色' }}
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isActive: true
    }
  },
  methods: {
    toggleColor() {
      this.isActive = !this.isActive
    }
  }
}
</script>
```



#### 3. 属性绑定（通过 data 变量）

将样式值提取到 `data` 中作为变量，便于统一管理和修改。

vue

```
<template>
  <div>
    <h3>3. 属性绑定</h3>
    <div :style="{ color: textColor, fontSize: textSize }">
      颜色：{{ textColor }}，字号：{{ textSize }}
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      textColor: 'green',
      textSize: '18px'
    }
  }
}
</script>
```



#### 4. 对象绑定（推荐）

将整个样式对象定义在 `data` 中，模板更简洁，且方便动态修改多个样式属性。

vue

```
<template>
  <div>
    <h3>4. 对象绑定</h3>
    <div :style="styleObject">
      样式由 styleObject 统一控制
    </div>
    <button @click="changeStyle">修改样式</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      styleObject: {
        color: 'red',
        fontSize: '15px',
        fontWeight: 'bold'
      }
    }
  },
  methods: {
    changeStyle() {
      this.styleObject.color = 'blue'
      this.styleObject.fontSize = '20px'
    }
  }
}
</script>
```
