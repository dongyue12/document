## 模版语法

### 返回字符

```vue
<template>
<div>
  <h1>名字</h1>
  <span>{{ name }}</span>
</div>
</template>

<script>
export default {
  data() {
    return {
      name: "张三",
    };
  },
}
</script>
```

### 返回数字

```vue
<template>
  <div>
    <p>数字是：{{ Number +1 }}</p>
  </div>
</template>
<script>
export default {
  data() {
    return {
      Number: 123
    };
  },
}
</script>
```

### 返回布尔值

```vue
<template>
  <div>
    <p>布尔是：{{ Boolean ? "是" : "否"}}</p>
  </div>
</template>
<script>
export default {
  data() {
    return {
      Boolean: true,
    };
  },
}
</scrip
```

### 返回html

主要是加入**v-html**才能识别

```vue
<template>
  <div>
    <p><span v-html="html"></span></p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      html: "<h1>我是h1</h1>",
    };
  },
}
</script>

```
