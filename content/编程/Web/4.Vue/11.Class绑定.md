## Class绑定

### 对象绑定

```vue
<template>
    <h3>Clss样式绑定</h3>
    <p :class="test">测试1</p>
    <p :class="{'active': active, 'error': error}">测试2</p>
    <p :class="{}">测试3</p>
</template>

<script>

export default {
    data() {
        return {
            test: 'red',
            active: true,
            error: true,
        }
    }
}
</script>

<style>
.red {
    color: red;
}
.active {
    font-size: 30px;
}
.error {
    font-weight: bold;
}
</style>
```

### 多个对象绑定

```vue

    <h3>Clss样式绑定</h3>
    <p :class="test">测试1</p>
    <p :class="{'active': active, 'error': error}">测试2</p>
    <p :class= 'Object' >测试3</p>
</template>

<script>

export default {
    data() {
        return {
            test: 'red',
            active: true,
            error: true,
            Object: {

                error: true

        }
    }
}
</script>

<style>
.red {
    color: red;
}
.active {
    font-size: 30px;
}
.error {
    font-weight: bold;
}
</style>
```

### 有条件的渲染样式

```vue
<template>
    <h3>Clss样式绑定</h3>
    <p :class="test">测试1</p>
    <p :class="{'active': active, 'error': error}">测试2</p>
    <p :class= 'Object' >测试3</p>
    <p :class="[test1, test2, test3]">测试4</p>
    <p :class="active ? 'active' : 'error'">测试5</p>

export default {
    data() {
        return {
            test: 'red',
            active: false,
            error: true,
            Object: {
                active: true,
                error: true
            },
            test1: 'red',
            test2: 'active',
            test3: 'error'
        }
    }
}
</script>

<style>
.red {
    color: red;
}
.active {
    font-size: 30px;
}
.error {
    font-weight: bold;
}
</style>
```

### 数组和对象一起用

```vue
<template>
    <h3>Clss样式绑定</h3>
    <p :class="test">测试1</p>
    <p :class="{'active': active, 'error': error}">测试2</p>
    <p :class= 'Object' >测试3</p>
    <p :class="[test1, test2, test3]">测试4</p>
    <p :class="active ? 'active' : 'error'">测试5</p>
    <p :class="[{'active': active}, test]">测试6</p>
</template>

<script>

export default {
    data() {
        return {
            test: 'red',
            active: true,
            error: true,
            Object: {
                active: true,
                error: true
            },
            test1: 'red',
            test2: 'active',
            test3: 'error'
        }
    }
}
</script>

<style>
.red {
    color: red;
}
.active {
    font-size: 30px;
}
.error {
    font-weight: bold;
}
</style>
```

> 数组和对象嵌套过程中，只能是数组嵌套对象
