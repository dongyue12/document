### Vue 侦听器（watch）与 v-model

#### 1. 基本概念

- **`v-model`**：实现表单输入与数据的**双向绑定**，输入变化时自动更新数据。
- **`watch`**：监听响应式数据的变化，并在变化时执行回调函数，适用于执行异步操作或开销较大的副作用。

#### 2. 示例代码

```vue
<template>
  <div>
    <h3>📡 侦听器示例（watch）</h3>

    <!-- 通过 v-model 双向绑定输入框内容 -->
    <input type="text" v-model="message" placeholder="输入文本" />

    <!-- 按钮通过方法修改数据，同样会触发 watch -->
    <button @click="changeMessage">更改为 "Hello Vue"</button>

    <p>当前消息：{{ message }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: 'Hello World'
    }
  },
  methods: {
    changeMessage() {
      this.message = 'Hello Vue'
    }
  },
  watch: {
    // 监听 message 数据的变化
    // 回调函数接收两个参数：新值（newValue）和旧值（oldValue）
    message(newValue, oldValue) {
      console.log(`message 发生变化：${oldValue} → ${newValue}`)
    }
  }
}
</script>
```

#### 3. 运行效果

- 在输入框中输入文字，控制台会实时打印新旧值的变化。
- 点击按钮将 `message` 改为 `"Hello Vue"`，控制台同样会输出变化记录。

### 侦听器的常用扩展

#### 3.1 深度监听（监听对象内部属性）

当需要监听**对象或数组内部值的变化**时，需使用 `deep: true` 选项。

```javascript
data() {
  return {
    user: {
      name: 'Alice',
      age: 25
    }
  }
},
watch: {
  user: {
    handler(newVal, oldVal) {
      console.log('user 对象内部属性变化')
    },
    deep: true    // 深度监听
  }
}
```

#### 3.2 立即执行（immediate）

默认情况下，侦听器只在数据变化时才执行。若希望**组件初始化时立即执行一次**，可设置 `immediate: true`。

```javascript
watch: {
  message: {
    handler(newVal, oldVal) {
      console.log('立即执行一次，且后续变化也会触发')
    },
    immediate: true
  }
}
```

#### 3.3 字符串方法名

若处理函数已在 `methods` 中定义，可以直接使用字符串形式引用。

```javascript
methods: {
  handleMessageChange(newVal, oldVal) {
    console.log('变化了')
  }
},
watch: {
  message: 'handleMessageChange'
}
```

### 计算属性 vs 侦听器 对比

| 特性         | 计算属性 (`computed`)          | 侦听器 (`watch`)                           |
| ------------ | ------------------------------ | ------------------------------------------ |
| **用途**     | 根据依赖数据**派生新数据**     | 观察数据变化并**执行副作用**（如异步请求） |
| **缓存**     | 自动缓存，依赖不变时不重新计算 | 无缓存，每次变化都执行回调                 |
| **返回值**   | 必须有返回值，作为属性使用     | 不需要返回值，执行操作即可                 |
| **异步支持** | ❌ 不支持异步                   | ✅ 非常适合异步操作                         |
| **使用方式** | `{{ computedValue }}`          | `watch: { dataName() {} }`                 |

### 补充：v-model 修饰符

在输入框中使用 `v-model` 时，可配合修饰符优化行为：

```html
<!-- .lazy：在 change 事件而非 input 事件后更新 -->
<input v-model.lazy="message" />

<!-- .number：自动将输入转为数字类型 -->
<input v-model.number="age" type="number" />

<!-- .trim：自动去除首尾空格 -->
<input v-model.trim="message" />
```
