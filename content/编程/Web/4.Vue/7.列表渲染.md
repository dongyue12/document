## 列表渲染（v-for）

### 1. 基本数组遍历

使用 `v-for="值 in 数组"` 遍历数组中的每一个元素。

```vue
<template>
  <h3>列表渲染 - 基本数组</h3>
  <div v-for="value in msg">{{ value }}</div>
</template>

<script>
export default {
  data() {
    return {
      msg: ['Hello World!', 'Hello Vue!', 'Hello JavaScript!']
    }
  }
}
</script>
```

> **说明**：`value` 是数组中每个元素的值，变量名可自定义。

---

### 2. 带索引的数组遍历

使用 `(value, index) in array` 获取当前元素及其索引（从 0 开始）。

```vue
<template>
  <h3>列表渲染 - 带索引</h3>
  <div v-for="(value, index) in msg">{{ value }} - {{ index }}</div>
</template>

<script>
export default {
  data() {
    return {
      msg: ['Hello World!', 'Hello Vue!', 'Hello JavaScript!']
    }
  }
}
</script>
```

---

### 3. 遍历对象数组（复杂数据）

当数组元素是对象时，可以访问对象的属性。

```vue
<template>
  <h3>列表渲染 - 对象数组</h3>
  <div v-for="item in result">
    <div>{{ item.name }}</div>
    <div>{{ item.price }}</div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      result: [
        { name: 'Vue', price: '一段文章' },
        { name: 'JavaScript', price: '一段文章' },
        { name: 'HTML', price: '一段文章' }
      ]
    }
  }
}
</script>
```

---

### 4. 遍历对象的属性

可以遍历普通对象的键值对，支持三个参数：`(value, key, index)`。

- `value`：属性值
- `key`：属性名
- `index`：遍历顺序的索引（从 0 开始）

```vue
<template>
  <h3>列表渲染 - 遍历对象</h3>
  <!-- 只获取值 -->
  <div v-for="value in userinfo">{{ value }}</div>

  <!-- 获取键、值、索引 -->
  <div v-for="(value, key, index) in userinfo">
    {{ key }} : {{ value }} : {{ index }}
  </div>
</template>

<script>
export default {
  data() {
    return {
      userinfo: {
        name: '张三',
        age: 18,
        sex: '男'
      }
    }
  }
}
</script>
```

**输出示例**：
```
name : 张三 : 0
age : 18 : 1
sex : 男 : 2
```

---

### 5. 绑定 key 值（重要）

Vue 在更新列表时，默认采用 **就地复用** 策略。为了准确追踪每个节点的身份，从而提高重排性能并维持组件状态，**必须为每个 `v-for` 项提供一个唯一的 `:key` 属性**。

```vue
<template>
  <h3>列表渲染 - 绑定 key</h3>
  <!-- 推荐：使用唯一标识（如 id） -->
  <div v-for="item in result" :key="item.id">
    {{ item.name }}
  </div>

  <!-- 仅当无唯一标识时，才使用索引（不推荐，可能导致状态错乱） -->
  <div v-for="(value, index) in msg" :key="index">
    {{ value }}
  </div>
</template>

<script>
export default {
  data() {
    return {
      result: [
        { id: 1, name: 'Vue' },
        { id: 2, name: 'JavaScript' }
      ],
      msg: ['A', 'B', 'C']
    }
  }
}
</script>
```

> **关键点**：
> - `:key` 是 `v-bind:key` 的简写，必须绑定一个**唯一且稳定**的值（如后端返回的 `id`）。
> - 不推荐使用 `index` 作为 key，因为当列表顺序变化时，Vue 可能会复用错误的元素，导致状态混乱。
> - 使用 `key` 可以帮助 Vue 区分不同元素，在重新排序时更高效地更新 DOM。

---

### 注意事项

1. **`v-for` 与 `v-if` 不推荐一起使用**  
   在同一个元素上同时使用 `v-for` 和 `v-if` 时，`v-for` 的优先级更高（Vue 2）或需要额外处理（Vue 3）。建议使用计算属性过滤数据。

2. **尽量使用 `key`**  
   不加 `key` 时 Vue 会发出警告，且列表更新可能产生非预期行为。

3. **遍历对象时，`index` 只是遍历顺序的索引**，不是对象的属性名或内部键。

4. **可以在 `<template>` 上使用 `v-for`** 来渲染多个元素而不引入额外 DOM 节点：
   ```vue
   <template v-for="item in list" :key="item.id">
     <div>{{ item.name }}</div>
     <div>{{ item.desc }}</div>
   </template>
   ```
