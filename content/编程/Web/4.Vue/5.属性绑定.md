## 属性绑定

### vue2的语法

选项式 API

```vue
<template>
  <div v-bind:id="dynamicId" v-bind:class="dynamicClass">AppID</div>
  <!-- 或者使用简写 -->
  <!-- <div :id="dynamicId" :class="dynamicClass">AppID</div> -->
</template>

<script>
export default {
  data() {
    return {
      dynamicId: "appid",
      dynamicClass: "appclass"
    }
  }
}
</script>
```



### vue3的语法



```vue
<template>
  <div :id="dynamicId" :class="dynamicClass">AppID</div>
</template>

<script setup>
import { ref } from 'vue'
const dynamicId = ref('appid')
const dynamicClass = ref('appclass')
</script>
```
