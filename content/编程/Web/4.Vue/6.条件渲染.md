## 条件渲染

### 条件渲染语句

- v-if
- v-else
- v-else-if
- v-show

#### V-if

```vue
<template>
<divv-if="flag">你能看见我么</div>
</template><script>
export default {dataO {
return
flag:true
子
</script>
```

#### v-else

```vue
<template>
   <div v-if="flag">if</div>
   <div v-else>v-else</div>
</template>

<script>
export default {
    data() {
        return {
            flag:false
        }
    }
}
</script>
```

#### v-else-if

```vue
<template>
   <div v-if="type === 'a'">a</div>
   <div v-else-if="type === 'b'">b</div>
   <div v-else>c</div>
</template>

<script>
export default {
    data() {
        return {
            flag: false,
            type: 'c'
        }
    }
}
</script>
```

#### v-show

v-show会在页面一直渲染，v-if则是只有条件为真才渲染

```vue
<template>
    <div v-show="flag">始终显示</div>
</template>

<script>
export default {
    data() {
        return {
            flag: true
        }
    }
}
</script>
```
